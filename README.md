# Grade Average Calculator - MOE Students

A responsive, web-based GPA calculator designed for MOE Students. This tool allows students to calculate their semester grade quickly and accurately directly from their browser, with no installation required.

**View Web Version :** *( https://adnanalbadawi.github.io/GradeCalculatorSchool/ )*
